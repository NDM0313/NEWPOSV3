# PWA Icons

Add these files for "Add to Home Screen" and app icon:

- **icon-192.png** (192×192 px)
- **icon-512.png** (512×512 px)

You can generate them from your logo (e.g. [realfavicongenerator.net](https://realfavicongenerator.net/) or [pwa-asset-generator](https://www.npmjs.com/package/pwa-asset-generator)).  
If icons are missing, the PWA may still be installable; the browser will use a default icon.

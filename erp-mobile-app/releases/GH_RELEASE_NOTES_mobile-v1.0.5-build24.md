# ERP Mobile 1.0.5 (build 24) — iOS Development IPA

**Bundle:** `com.dincouture.erp`  
**API:** `https://erp.dincouture.pk`  
**Export:** Development (install via Xcode → Devices and Simulators)

## What’s new

- Cash Flow: full list + account filter chips (see `CASH_FLOW_ACCOUNT_FILTER.md`)
- Branch default cash/bank on payment pickers; newest-first report defaults
- Fresh prod Capacitor sync

## Install

1. Download `erp-mobile-1.0.5-build24.ipa` from this release.
2. Xcode → Window → Devices and Simulators → select your registered iPhone → drag the IPA onto the device.

Device must be registered to Apple team **NLNZN84GX4**.

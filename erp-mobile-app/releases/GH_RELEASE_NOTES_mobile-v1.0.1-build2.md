## ERP Mobile 1.0.1 (versionCode 2)

Signed Release APK for internal / counter tablet distribution.

### Highlights

- Shared counter / POS lock screen (enrolled users, boot and logout lock).
- **Counter tablet PIN** in Settings for **all staff** with a concrete branch.
- **Unique 4-digit PIN** per person on the tablet (duplicate PIN no longer overwrites another user).
- First-login **Set PIN** can enroll counter vault when PIN is 4 digits and branch resolves.

### Source

- Main commits: `9912cfe` (feature), `65733a8` (APK log SHA line).
- Full changelog: `erp-mobile-app/releases/APK_UPDATE.md` on `main`.

### Install

- Uninstall old debug builds if the package signature conflicts with this release build.

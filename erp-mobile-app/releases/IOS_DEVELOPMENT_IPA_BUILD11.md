## Build commands (Mac)

From repo root:

```bash
cd erp-mobile-app
npm ci
# Ensure .env has VITE_SUPABASE_URL + VITE_SUPABASE_ANON_KEY (production)
npm run cap:sync:ios:prod

cd ios/App
xcodebuild -scheme "NDM ERP" \
  -project App.xcodeproj \
  -configuration Release \
  -destination 'generic/platform=iOS' \
  -archivePath ../../build/NDM-ERP-build11.xcarchive \
  archive

xcodebuild -exportArchive \
  -archivePath ../../build/NDM-ERP-build11.xcarchive \
  -exportPath ../../releases/ios-export-build11 \
  -exportOptionsPlist ../../releases/ios-export-build11/ExportOptions.plist

cp "../../releases/ios-export-build11/NDM ERP.ipa" "../../releases/erp-mobile-1.0.5-build11.ipa"
ls -lh "../../releases/erp-mobile-1.0.5-build11.ipa"
```

## Install on iPhone

1. Uninstall older **NDM ERP** if install fails (profile/signature change).
2. **Xcode → Window → Devices and Simulators** → select device → drag `erp-mobile-1.0.5-build11.ipa`.
3. Or use **Apple Configurator**.
4. **Settings → General → VPN & Device Management** → trust developer if prompted.
5. Device UDID must be on team **NLNZN84GX4** (Development export).

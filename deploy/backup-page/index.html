Database Backups

Self-hosted project – scheduled backups via server script.

Scheduled backups Point in time Restore to new project

Backups are created on the server. Run the command below on your VPS to generate a full database dump. Optionally schedule it with cron for daily backups.

📅
Self-hosted backup command

Run on the server (e.g. ssh root@your-vps) to create a backup. Retention = last argument (days).

cd /root/NEWPOSV3 && bash deploy/backup-supabase-db.sh 7

Backups are saved under ./backups/supabase_db_YYYYMMDD_HHMMSS.dump. Daily cron example: 0 2 * * * cd /root/NEWPOSV3 && bash deploy/backup-supabase-db.sh 14

You can also use ERP → Settings → Data & Backup for the same instructions.

# Post–go-live monitoring checklist (NEW BUSINESS)

Use this during the **first live session** on NEW BUSINESS. No new features — only observe that core flows work.

- [ ] **First dashboard open** — Dashboard loads; no console errors.
- [ ] **First sale** — Create one sale (draft or final); save; view in list.
- [ ] **First purchase** — Create one purchase; save; view in list.
- [ ] **First accounting page open** — Accounting module opens; Journal / Daybook / Roznamcha load.
- [ ] **First report open** — Open Trial Balance or P&L; report runs without error.
- [ ] **First courier/shipment check** — Accounting → Courier tab opens; shipment_ledger data (if any) or empty state.
- [ ] **First rental open** (if used) — Rental list/dashboard opens.
- [ ] **First studio open** (if used) — Studio list/dashboard opens.

If any item fails, note it and fix before continuing normal use.
